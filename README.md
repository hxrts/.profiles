profiles
========

for syncing profile settings across machines

## installation prep

- git
- zsh
- oh-my-zsh
- tmux
- nvim
- pathogen
- emacs
- rvm
- miniconda
- tree
- R


## instructions

1. `git clone https://github.com/hxrts/.profiles`

2. `git submodule update --recursive --remote`

3. sh ~/.profiles/init.sh
